"New for enterprises: OpenAI Presence helps companies deploy trusted voice and chat agents across customer and internal workflows. AI agents can answer questions, use company systems, take approved actions, and escalate to people when needed—while improving over time."

02 工具更新

Grok Build 新增 Exa 插件：语义搜索与多步研究

"Grok Build just got another powerful plugin: Exa. The new Exa plugin brings semantic web search, content extraction and multi-step research directly into Grok Build. ... No API key is required, and new accounts receive free credits at signup."

实用点

Exa 插件将语义搜索（非关键词匹配）引入 Grok Build 工作流，支持按公司、人物、新闻、论文、财报过滤结果，并输出 Markdown。对于 agent 开发者，这意味着可以在同一 AI 工作流中完成研究 + 开发，无需切换工具。设置简单：安装插件 → 浏览器登录 Exa 账户 → 开始使用。

Pieter Levels 发布 "AI 驱动服务器" 完整部署指南：从 VPS 到 Claude Code

"If you wanna do it yourself, this is how: Buy (~15 min) ... Lock it down (~20 min) ... Install the brain (~5 min) ... Hand over everything else ... From then on you never administer the server again. You open Termius from anywhere, on any device, and just say what you want."

实用点

这是一份面向开发者的完整实操指南，展示了如何将一台 €8/月的 Hetzner VPS 变成完全由 Claude Code 管理的 "AI 服务器"。关键步骤包括：Tailscale 私有网络 + 防火墙锁定（仅允许 Cloudflare IP）、SSH 密钥管理、tmux 持久会话、CLAUDE.md 手写提示词。对于 CS/agent 开发者，这是 "AI 作为运维工程师" 的实战案例。

06 关注动态

OpenAI 发布 Contrastive SDF：测量模型 "奖励寻求" 行为的新方法

"Contrastive SDF gives copies of the same model opposing beliefs about what the grader prefers, then measures how their behavior changes. ... We had guessed reward seeking might increase over the course of capabilities-focused RL training, but had no way of measuring it until now."

实用点

这是 OpenAI 与 Apollo Research 合作的延续研究。Contrastive SDF 是一种新的测量方法，通过给同一模型的不同副本灌输相反的 "评分者偏好" 信念，观察其行为变化，从而量化模型的 "奖励寻求" 倾向。对于 AI 安全研究者，这是一个新的评估工具。
